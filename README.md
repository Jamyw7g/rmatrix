# RMatrix

Rmatrix is similar to the [cmatrix](https://github.com/abishekvashok/cmatrix), but it wrote it in `Rust`. Get up to 75% performance improvement compared to cmatrix.

## Build and Install

**Minimum Rust requires 1.53.0**

```shell
# build
cargo build --release

# install
cargo install --path .
```
